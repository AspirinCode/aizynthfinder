class aizynthfinder.mcts.config.Configuration¶

Bases: object

Encapsulating the settings of the tree search, including the policy, the stock and various parameters.

All the parameters can be retrieved as attributes of the Configuration object, e.g.

config.max_transforms  # The maximum of transform
config.iteration_limit  # The maximum number of iterations

On instantiation it will read default parameters from a config.yml file located in the data folder of the package.

class aizynthfinder.mcts.node.Node(state, owner, config, parent=None)¶

Bases: object

A node in the search tree.

The children are instantiated lazily for efficiency: only when a child is selected the reaction to create that child is applied.

Properties of an instantiated children to a node can be access with:

children_attr = node[child]

the return value is a dictionary with keys “action”, “value”, “prior” and “visitations”.

Variables

  • is_expanded (bool) – if the node has had children added to it

  • parent (Node) – the parent node

  • is_expandable (bool) – if the node is expandable

  • state (State) – the internal state of the node

Parameters

  • state (State) – the state of the node

  • owner (SearchTree) – the tree that owns this node

  • config (Configuration) – settings of the tree search algorithm

  • parent (Node, optional) – the parent node, defaults to None

Individual policies are dictionaries with the keys “model” and “templates”, representing the loaded Keras model and the loaded template dataset.

class aizynthfinder.mcts.state.State(mols, config)¶

Bases: object

Encapsulation of the molecular state of a node.

A state consists of an immutable list of molecules that are either solved (can be found in stock) or that potentially can be expanded to new molecules by applying a reaction on them.

The class is hashable and comparable by the inchi keys of all the molecules.

Variables

  • mols (list of TreeMolecule) – the list of molecules

  • stock (Stock) – the configured stock

  • policy (Policy) – the configured policy

  • in_stock_list (list of bool) – for each molecule if they are in stock

  • is_solved (bool) – is true if all molecules are in stock:

  • max_transforms (int) – the maximum of the transforms of the molecule

  • is_terminal (bool) – is true if the all molecules are in stock or if the maximum transforms has been reached

Parameters

  • mols (list of TreeMolecule) – the molecules of the state

  • config (Configuration) – settings of the tree search algorithm

class aizynthfinder.mcts.stock.InMemoryInchiKeyQuery(filename)¶

Bases: object

A stock query class that is based on an in-memory list of pre-computed inchi-keys.

This list can be instantiated from

  • A Pandas dataframe in HDF5 or CSV format

  • A text file with an inchi key on each row

The pandas dataframe must have a column “inchi_key” with InChIkeys. The HDF5 file must have a dataset called “table”.

Parameters

filename (str) – the path to the file with inchi-keys

class aizynthfinder.mcts.stock.Stock¶

Bases: object

A collection of molecules that are in stock

A molecule can be queried on the stock with:

my_mol = Molecule(smiles="CCO")
my_mol in stock

One can obtain individual stocks with:

sub_stock = stock["key"]

One can obtain the number of molecules in the selected stock with:

number_of_molecules = len(stock)
